MCP Server · Online

Portfolio MCP Server

An MCP server exposing the portfolio data of Al Hinds — blog posts, work experience, qualifications, and profile information.

Endpoint
https://mcp.alhinds.com/mcp POST
Quick Start

Add the following to your MCP client config — works with Claude Desktop, Cursor, and any MCP-compatible client.

claude_desktop_config.json 
{
  "mcpServers": {
    "portfolio": {
      "url": "https://mcp.alhinds.com/mcp"
    }
  }
}
Available Tools
get_posts
Get all blog posts, optionally filtered by tag
get_post
Get a specific blog post by its slug
get_jobs
Get all work experience and jobs
get_qualifications
Get all education and qualifications
get_user
Get user profile information including socials
get_resume
Get a formatted resume with jobs, qualifications, and user info
